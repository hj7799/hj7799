# GeoForge GIS

一个基于浏览器的地理信息系统，无需安装任何软件。

🌐 **在线访问：** [https://hj7799.github.io/HJ7799](https://hj7799.github.io/HJ7799)

## 功能

- 🗺 地图可视化（6种底图 + 卫星影像）
- 📂 本地数据加载（GeoJSON / KML / CSV / Shapefile）
- 📐 空间分析（缓冲区、相交、合并、质心、面积、距离）
- 🔄 格式转换（GeoJSON / KML / CSV / WKT / TopoJSON）
- 📍 地理编码（地址 ↔ 经纬度）

## 技术栈

- [Leaflet.js](https://leafletjs.com/) — 地图渲染
- [Turf.js](https://turfjs.org/) — 空间分析
- [Nominatim](https://nominatim.org/) — 地理编码
- [Esri World Imagery](https://www.esri.com/) — 卫星影像
